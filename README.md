### 📂 Featured Projects

#### 🎮 [**HELLO IT HEAR ME**](https://github.com/Gaer12TH)
An immersive desktop-simulation survival game developed in Unity C#. Features custom UI management systems, time-scale manipulation, dynamic storytelling through intro phone calls, and complex program window animations.

#### 🎛️ [**Antigravity Multi-ID Dashboard**](https://github.com/Gaer12TH)
A comprehensive full-stack dashboard built with React, Vite, Tailwind CSS, and Vercel KV (Redis) with Serverless Functions. Handles multiple account management, automated token extraction with a Python GUI, and local SQLite data synchronization.

#### 🐺 [**Werewolf Moderator Tool**](https://github.com/Gaer12TH)
A single-page static web application designed as a companion tool for managing local Werewolf games. Built with HTML, JS, and Tailwind CSS. Highlights include role presets stored in localStorage, Night/Day phase tracking, and built-in discussion timers.
